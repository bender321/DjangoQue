# DjangoQue

### Interpreter version:
- Python >= 3.10

# Instruciton
- Clone this repository
- create venv + install all requirements
- move to directory "quesite"
- Run command ```python manage.py runserver```

- go to 127.0.0.1:8000/que
- try the app
